EclipseLink's Reference Implementation Downloads

EclipseLink is a leader is providing standards based open source persistence solutions. The following downloads are reference implementations delivered by the EclipseLink project.

Jakarta Persistence API 3.1

The EclipseLink JPA component in EclipseLink provides the compatible implementation of Jakarta Persistence 3.1.

Downloads:

  • EclipseLink 4.0.8 - Full EclipseLink build, including the Jakarta Persistence 3.1 CI (TCK Summary).

    Jakarta Persistence API 3.0

    The EclipseLink JPA component in EclipseLink provides the compatible implementation of Jakarta Persistence 3.0.

    Downloads:

    • EclipseLink 3.0.4 - Full EclipseLink build, including the Jakarta Persistence 3.0 CI (TCK Summary).

      Java Persistence API 2.2 (JSR 338)

      The EclipseLink JPA component in EclipseLink provides the reference implementation of JPA 2.2.

      Downloads:

      • EclipseLink 2.7.16 - Full EclipseLink build, including the JPA 2.2 RI/CI (TCK Summary).

        Java Persistence API 2.1 (JSR 338)

        The EclipseLink JPA component in EclipseLink provides the reference implementation of JPA 2.1.

        Downloads:

        • EclipseLink 2.6.9 - Full EclipseLink build, including the JPA 2.1 RI.
        • minimal bundles containing the JPA 2.1 RI

          Java Persistence API 2.0 (JSR 317)

          The EclipseLink JPA component in EclipseLink provides the reference implementation of JPA 2.0.

          Downloads:

          • EclipseLink 2.0.0 - Full EclipseLink build, including the JPA 2.0 RI.
          • EclipseLink 2.0.0 - minimal bundles that are shipped in the Java EE6 RI.

            Service Data Objects 2.1.1 (JSR 235)

            The EclipseLink SDO component in EclipseLink 1.1 provides the reference implementation of SDO 2.1.1.

            Downloads:

            • EclipseLink 1.1.0 - Complete
            • EclipseLink 1.1.0 - Only the SDO 2.1.1 required components
