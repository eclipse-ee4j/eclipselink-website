Preface

Oracle TopLink provides specific annotations (TopLink extensions) in addition to supporting the standard Java Persistence Architecture (JPA) annotations. You can use these TopLink extensions to take advantage of TopLink's extended functionality and features within your JPA entities.

Audience

This document is intended for application developers who want to develop applications using Oracle TopLink with Java Persistence Architecture (JPA). This document does not include details about related common tasks, but focuses on Oracle TopLink functionality.

Developers should be familiar with the concepts and programming practices of

  • Java SE and Java EE.

  • Java Persistence Architecture 2.0 specification (http://jcp.org/en/jsr/detail?id=317)

  • Eclipse IDE (http://www.eclipse.org)
