@StructConverters

Use @StructConverters to define multiple @StructConverter annotations.

Annotation Elements

Table 2-61 describes this annotation's elements.

Table 2-61 @StructConverters Annotation Elements

Annotation Element Description Default

StructConverter[]

(Required) An array of struct converter

Examples

Example 2-91 shows how to use the @StructConverters annotation to define multiple @StructConverter elements.

Example 2-91 Using @StructConverters Annotation

@StructConverters{{
  @StructConverter(name="StructConverter1", converter="foo.StructConverter1"),
  @StructConverter(name="StructConverter2", converter="foo.StructConverter2")
  })

Example 2-92 shows how to use the <struct-converters> element in the eclipselink-orm.xml file.

Example 2-92 Using <struct-converters> XML

<struct-converters>
  <struct-converter name="StructConverter1" converter="foo.StructConverter1"/>
  <struct-converter name="StructConverter2" converter="foo.StructConverter2"/>
</struct-converters>
